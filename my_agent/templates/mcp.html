接入说明

stdio 传输（推荐本地开发）
  1. 复制 stdio 配置到 ~/.cursor/mcp.json
  2. 修改 cwd 为项目实际路径
  3. 填入 LLM_API_KEY
  4. 重启 Cursor，在 Agent 模式下即可使用 MyAgent 工具
SSE 传输（推荐远程/生产）
  1. 启动 MyAgent 服务（uvicorn my_agent.main:app）
  2. 复制 SSE 配置到 ~/.cursor/mcp.json
  3. 修改 URL 为实际服务地址
  4. 重启 Cursor 即可连接
